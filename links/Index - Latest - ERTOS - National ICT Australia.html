Project: Iguana

Iguana is designed as a base for the provision of operating system (OS) services for embedded systems. Among others, it provides the underlying OS for Wombat, our version of paravirtualised Linux designed to provide legacy support for embedded systems. To get started playing with it read our getting started page. For more information see the overview.
